# NebulaX Project Brief

## Project Overview
NebulaX is a comprehensive cryptocurrency trading platform that integrates with Hyperliquid for live market data, trading functionality, and portfolio management. The platform provides professional-grade trading tools including real-time charts, order books, and trading panels.

## Core Objectives
- Provide live cryptocurrency trading with Hyperliquid integration
- Offer professional trading interface with real-time market data
- Support multiple trading pairs and timeframes
- Implement secure wallet integration and order management
- Deliver responsive, mobile-friendly trading experience

## Key Features
- Real-time price feeds from Hyperliquid
- Live order book display
- Interactive trading charts with multiple timeframes
- Portfolio tracking and management
- Secure authentication and wallet integration
- Mobile-responsive design

## Technical Stack
- Frontend: React, TypeScript, Tailwind CSS
- Backend: Node.js, Express, TypeScript
- Database: PostgreSQL with Drizzle ORM
- Market Data: Hyperliquid API integration
- Charts: TradingView Lightweight Charts
- Authentication: Custom wallet-based auth

## Current Status
The platform is actively under development with core trading components implemented and Hyperliquid integration completed for live data feeds.
